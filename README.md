# DIY-Rotoscoping-Style-Cartoon-Portrait
This is a code that converts portrait images into rotoscoping animation style using OpenCV filters.

본 프로젝트는 OpenCV 라이브러리를 활용하여 일반 실사 이미지를 만화스타일로 변환하는 Python 스크립트입니다.

## 💡 구현 파라미터 튜닝 (알고리즘 핵심)
강의에서 학습한 필터들을 조합하여 최적의 카툰 렌더링 결과물을 도출하기 위해 다음과 같이 파라미터를 조정했습니다.
1. **스케치 마스크 생성 (`cv2.adaptiveThreshold`):** 기본 상수 $C$ 값을 `5`로 설정하여 윤곽선이 중간에 끊기지 않고 굵고 거친 클래식 코믹북 스타일의 펜 터치가 나오도록 하였습니다.
2. **색상 평활화 (`cv2.bilateralFilter`):** 엣지를 보존하면서 내부 색상을 단순화하기 위해 `d=9, sigmaColor=300, sigmaSpace=300`의 강력한 필터를 적용하여 수채화 같은 질감을 연출했습니다.

## 🔍 한계점 및 논리적 분석 (Limitations Analysis)

위의 실패 사례가 발생한 원인을 컴퓨터 비전 관점에서 분석하고, 이를 극복하기 위해 전략을 수정한 과정입니다.

### 1. 망원 렌즈 사진의 심도 표현 한계
카메라는 빛과 물리적 거리(Depth)를 담아내어 아웃포커싱(얕은 심도)을 구현하지만, 컴퓨터 비전 알고리즘(`adaptiveThreshold`)은 사진을 원근감이 거세된 납작한 2D 평면상의 픽셀 배열로만 인지합니다. 이로 인해 알고리즘은 배경의 부드럽게 뭉개진 빛망울이나 미세한 명암 차이조차 모두 '윤곽선(Edge)'으로 오인하여 시커먼 펜 터치를 그어버리는 근본적인 한계가 있었습니다. 

### 2. 전략 수정 및 한계 극복
컴퓨터 비전 알고리즘이 2D 평면의 픽셀 대비(Contrast)에만 민감하게 반응한다는 한계를 극복하기 위해 대상 이미지를 변경하는 전략을 취했습니다. 복잡한 심도 표현이 들어간 배경 대신, 배경이 단순하고 피사체의 명암 대비가 명확한 인물 사진으로 렌더링 대상을 바꿨습니다. 그 결과, 원하지 않는 노이즈 선들이 사라지고 팝아트/로토스코핑 기법 스타일의 매력적인 초상화 렌더링을 성공적으로 가공할 수 있었습니다.
